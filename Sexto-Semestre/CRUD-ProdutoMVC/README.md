# CRUD Produtos MVC

## Sobre o projeto

O projeto foi desenvolvido no 6º semestre do curso, durante a disciplina de Tópicos Especiais de Informática, cursada no primeiro semestre de 2026 e ministrada pelo professor Júlio Fernando Lieira.

A aplicação consiste em um sistema web para gerenciamento de produtos, permitindo realizar as operações de cadastro, consulta, atualização e exclusão (CRUD). O desenvolvimento teve como foco a aplicação dos conceitos de Programação Orientada a Objetos (POO) utilizando PHP, organizando a aplicação seguindo o padrão de arquitetura MVC (Model-View-Controller).

### Objetivo

O objetivo foi desenvolver um sistema de gerenciamento de produtos utilizando os principais conceitos estudados durante a disciplina, como:

- **Programação Orientada a Objetos (POO)**: aplicação dos três pilares da orientação a objetos (Encapsulamento, Herança e Polimorfismo).
- **Arquitetura MVC**: separação das responsabilidades entre Model, View e Controller, tornando o código mais organizado e de fácil manutenção.
- **Fundamentos do PHP**: utilização da linguagem para construção da lógica de negócio da aplicação.
- **Conexão com banco de dados**: utilização do PHP para estabelecer a comunicação com o MySQL, realizando operações de persistência e recuperação de dados.
- **Controle de sessão**: gerenciamento da autenticação e persistência de informações durante a navegação.
- **URL amigável**: implementação de rotas mais legíveis e organizadas para acesso às páginas da aplicação.
